# 🚗 Ride Booking App (Uber Clone)

A modern Ride Booking mobile application built using **Flutter**, featuring smooth animations, clean UI, and a complete ride flow including booking, driver tracking, and payment screens.

This project is designed for **learning, portfolio building, and UI/UX inspiration**.

---

## ✨ Features

* 🔐 Authentication (Login / Signup UI)
* 📍 Live Location UI (Map Placeholder)
* 🔎 Search Pickup & Drop Location
* 🚘 Ride Selection (Mini, Sedan, SUV)
* 💰 Fare Estimation
* 👨‍✈️ Driver Assignment Screen
* 📡 Live Ride Tracking (Mock Animation)
* 💳 Payment Screen (Cash / UPI UI)
* 📜 Ride History
* 👤 Profile Screen
* 🌙 Dark Mode Support
* 🎬 Smooth Animations (Hero, Fade, Slide)

---

## 🛠️ Tech Stack

* **Flutter**
* **Dart**
* **Provider / Riverpod (State Management)**
* **Material UI / Custom UI**
* **Lottie Animations**
